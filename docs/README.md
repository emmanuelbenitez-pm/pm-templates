### By Project Phase
- **Initiation:** Project Charter, Business Case
- **Planning:** WBS, Schedule, Budget, Risk Register
- **Execution:** Status Reports, Issues Log, Meeting Notes
- **Monitoring:** Change Requests, RAID Log
- **Closure:** Lessons Learned, Transition Plan

### By Methodology
- **Waterfall:** Traditional sequential templates
- **Agile/Scrum:** Sprint planning, retrospectives
- **Hybrid:** Mix of both approaches
