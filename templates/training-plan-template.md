## ğŸ“– Training Curriculum

### Course 1: End User Fundamentals (4 hours)

**Target Audience:** All end users (400 people)  
**Delivery Method:** Virtual instructor-led  
**Class Size:** 20-25 per session  
**Prerequisites:** None

**Course Outline:**

**Module 1: Introduction (30 min)**
- Why we're implementing this system
- Benefits for sales reps
- Overview of interface
- How to get help

**Module 2: Navigation Basics (30 min)**
- Logging in (SSO)
- Home page and dashboards
- Main navigation menu
- Search functionality
- Mobile app overview

**Module 3: Customer Management (60 min)**
- Creating customer records
- Searching for customers
- Viewing customer information
- Updating customer data
- Activity: Create and update 3 customer records

**Module 4: Opportunity Management (60 min)**
- Creating opportunities
- Moving opportunities through stages
- Updating opportunity details
- Associating activities with opportunities
- Activity: Create opportunity, log activity

**Module 5: Activity Tracking (30 min)**
- Logging calls
- Logging meetings
- Email integration
- Setting tasks and reminders
- Activity: Log 3 different activity types

**Module 6: Reporting (20 min)**
- Running standard reports
- Exporting data
- Reading dashboards
- Activity: Generate and export report

**Module 7: Mobile Access (10 min)**
- Downloading mobile app
- Key mobile features
- Offline access
- Demo: Mobile walkthrough

**Module 8: Wrap-up (10 min)**
- Q&A
- Resources (help docs, videos)
- Support contacts
- Feedback survey

**Hands-on Practice:** 50% of class time  
**Assessment:** End-of-session quiz (10 questions)

### Kirkpatrick's Four Levels of Evaluation

#### Level 1: Reaction (Did they like it?)
**Measurement:** End-of-training survey (5-point Likert scale)

**Questions:**
- Overall training quality: ___/5
- Instructor effectiveness: ___/5
- Materials quality: ___/5
- Hands-on practice value: ___/5
- Pace appropriateness: ___/5
- Would recommend to others: Yes/No

**Target:** Average >4.0/5.0 across all questions

---

#### Level 2: Learning (Did they learn?)
**Measurement:** End-of-training assessment (20 questions)

**Question Types:**
- Multiple choice (10): Knowledge recall
- True/False (5): Concepts
- Scenario-based (5): Application

**Target:** >85% pass rate (17/20 correct)

**Sample Questions:**
1. How do you create a new customer record? [Multiple choice]
2. What is the first stage in the sales process? [Multiple choice]
3. You need to log a customer meeting. Describe the steps. [Scenario]

---

#### Level 3: Behavior (Are they using it?)
**Measurement:** System usage analytics + observation

**Metrics Tracked:**
- Login frequency (target: 80% daily active users)
- Features used (target: 8/10 core features)
- Data quality (target: >95% complete records)
- Task completion time (target: <2 min average)

**Measurement Period:** 30, 60, 90 days post-training

**On-Floor Observation:**
- Super users observe 10 users each
- Complete observation checklist
- Provide coaching as needed
